# Disclaimer / TLDR

This library is a hobby/passion project that I make for fun in order to cut boilerplate and solve issues I usually have.
Whichever way any part of it was made, the result is mine and for my own usage.
If AI can make it better and/or faster, then I will use it.

NoireLib comes with no warranty, if you are unsure about any of this, then please do not use it.

# AI Usage Disclosure

The foundations and the architecture of this library were written by a human.
So are the system design, the library philosophy, and the thinking process behind the UX.

I clearly define what I want and what direction it should take, AI then brainstorms around it.
It may propose improvements and show implementation examples, and I then approve, change, or refuse them.

Editor and Copilot autocompletion was used.

Past that, the level of AI involvement varies by area. Some systems were only assisted; others were
built largely by AI from my direction, Draw3D being the clearest example.

All of the documentation READMEs are AI-written, and most of the docblocks as well.
The unit tests are AI-written, and they are more of a regression checking tool rather than a proof of
correctness.

Not every single line of code has been reviewed or authored, but I actively try to improve the
library as I use it inside plugins. If I use NoireLib in a plugin, it means that I've read what was used.

Most of the modules, helpers, and systems have been tested by hand in a private test plugin, as
best as I can. Exceptions are systems requiring another OS (Linux, macOS) or a second PC.
AI periodically checks over the code looking for errors and inconsistencies.
